# AGENTS.md — J-TRINITY Documentation Website Rules

This repository is for an educational documentation website derived from the J-TRINITY doctrine.

## Hard Rules

- This is a learning website only.
- Preserve SIMULATION_ONLY / ADVISORY_ONLY / HUMAN_FINAL language.
- Do not build operational command workflows.
- Do not build weapon, drone, vehicle, GPS-control, actuator, or external-control paths.
- Do not introduce real-world execution features.
- Do not convert advisory concepts into command language.
- Keep content modular and Markdown-first.
- Prefer Docusaurus-compatible Markdown.
- Keep diagrams educational and conceptual.

## Website Goal

Create a professional online textbook / digital doctrine library for J-TRINITY C4ISR.

## Design Direction

- Tactical academic style
- Dark green / black / slate theme
- High readability
- Sidebar navigation
- Glossary
- Search
- Diagrams for JD / JM / JB / JG / HUMAN, Event Timeline, Trust, Conflict, Guardian, AAR
